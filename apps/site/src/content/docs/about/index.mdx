<h2 id="sustainability">Sustainability</h2>

**This site is maintained by one person.** If something breaks, expect 24–72 hours to a fix during normal operations. During declared sustainability mode (banner visible site-wide), expect a weekly cadence instead of daily. If maintenance pauses for longer than two weeks, the README will say so. <a href={`${REPO_URL}/issues/new?labels=co-maintainer-interest`}>Co-maintainer wanted →</a>

See <a href={`${REPO_URL}/blob/main/MAINTENANCE.md`}>MAINTENANCE.md</a> for the full operational model: weekly review cap, escalation path, graceful sunset plan.

## How to contribute

See <a href={`${REPO_URL}/blob/main/CONTRIBUTING.md`}>CONTRIBUTING.md</a>. Briefly: edit the Markdown, follow the voice rules (no "you should", no marketing words), open a PR. The site has zero tracking and never will. No cookies, no analytics that retain PII, no third-party CDN.
